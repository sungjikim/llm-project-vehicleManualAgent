🔧 주요 특징 (성능 최적화 버전)

  • 조건부 워크플로우: 응급 상황 감지 후 적절한 경로 자동 선택
  • 응급 상황 빠른 경로: CRITICAL/HIGH 응급 상황 전용 최적화된 처리
  • 간소화된 검색: 응급 상황에서 BM25 키워드 검색만 사용
  • 최적화된 응급 감지: CRITICAL/HIGH 키워드 우선 검사
  • 고정 신뢰도: 응급 상황에서 복잡한 평가 생략
  • SubGraph 아키텍처: 각 기능을 독립적인 SubGraph로 모듈화
  • 하이브리드 검색: 일반 질문에서는 벡터 검색과 키워드 검색 결합
  • 주행 중 최적화: 운전 중 상황을 감지하여 답변을 압축
  • 실시간 모니터링: 성능 및 사용량을 실시간으로 추적
  • 다중 인터페이스: 터미널과 웹 인터페이스 모두 지원

🎯 LLM 기반 감지의 핵심 특징

  • 🧠 맥락 이해: 키워드 매칭이 아닌 의미적 분석으로 정확한 판단
  • 📊 신뢰도 제공: 각 판단에 대한 신뢰도 점수와 근거 설명
  • 🎯 정확한 분류: "브레이크 패드 관리방법" → 일반 질문으로 정확 분류
  • ⚡ 성능 최적화: 응급 질문 응답시간 77.8% 향상 (11.5초 → 2.55초)
  • 🔄 자동 적응: 새로운 상황 패턴에 자동으로 적응
  • 🚗 주행 감지: LLM 기반 주행 중 상황 자동 감지 및 답변 압축
  • 📱 스마트 압축: 주행 중 안전을 위한 핵심 정보만 제공
  • 🔍 하이브리드 검색: 벡터 + BM25 검색의 조합
  • 📈 실시간 모니터링: 성능 지표 및 신뢰도 추적
  • 🎤 음성 지원: ASR/STT 기반 음성 입력 처리
